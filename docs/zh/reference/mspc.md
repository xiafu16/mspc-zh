  珠海海洋科普中心(以下简称 “中心”)坐落于南方海洋实验室所在的中山大学珠海校区海琴四号楼，展示面积达3707平方米。中心以南方海洋实验室的重要基础设施和科研成果为依托，联合中山大学的优质科教资源，共同打造湾区海洋科普重要阵地。

  中心实现了“科普、科教、科研”的一体化融合，充分彰显海洋学科特色、珠海地市特色以及科技创新特色。中心以海洋科学知识为脉络，从“海洋观测”角度切入,讲述 “海洋发现之旅”的故事,共分为四大章节。参观内容涉及海洋科学的诞生、海洋观测新技术、海水的组成与运动、神奇的海底地形地貌及生态系统、珠海海洋特色等知识点，并且重点选取南方海洋实验室的科研成果--全球首艘无人智能科考母船 “珠海云” 作为体验展项，使观众能够获得科考船驾驶的实训级体验。

  中心致力于提供多元化、全方位的海洋科普教育体验，与实验室现有的展示中心、各类先进的公共实验平台以及多功能讲厅相结合，统筹形成标准化、专业化、特色化相结合的科普教育体系，以实现常态化海洋科普教育和特色科普研习两大功能。

  作为南方海洋实验室海洋科普教育基地的重要组成部分，中心以“携手共圆海洋梦”为服务理念，以“弘扬科学精神、点燃科学梦想”为重要使命，将努力提高社会各界人士“认识海洋、关心海洋、经略海洋”的意识与能力，矢志成为启迪创新智慧、厚植科学素养、弘扬海洋文化的粤港澳大湾区科普教育新标杆。
