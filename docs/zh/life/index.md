# 海洋生物

海洋生物是指生活在海洋或沿海河口咸水区域的植物、动物和其他生物。海洋生物能产生氧气和固碳。

地球上有许多陆上生物是由海洋生物演化而来。例如脊椎动物起初只在水中生活，后来其中的一些物种演化成两栖动物。还有其他一些鱼类则演化成哺乳动物，并来到陆地上居住，不过有一些又回到海洋中，例如海豹、海豚和鲸鱼。浮游生物则是海洋生物食物链的最底端。

有一些无脊椎动物为了能在含氧量低的海洋中生存而演化出独特的构造，例如软体动物依靠虹管呼吸，鱼类则通过鳃呼吸。一些海洋哺乳动物，如海豚、鲸鱼、海獭和海豹，需要定期浮出水面呼吸空气。

已知海洋生物有20多万种，可能还有200多万种海洋生物尚待人类发现。海洋生物大小不一，像浮游植物仅长0.02微米，而蓝鲸则长达33米。海洋微生物，包括原生动物、细菌和病毒，占海洋生物总量的70%。
